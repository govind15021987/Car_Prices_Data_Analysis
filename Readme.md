# Car Prices Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Car Prices dataset using Python, Pandas, Matplotlib and Seaborn.

## Tasks Covered
- Data Cleaning
- Missing Value Treatment
- Duplicate Removal
- Statistical Analysis
- Data Visualization
- Business Insights

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
